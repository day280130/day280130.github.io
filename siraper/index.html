    Description

    Best Customer Recommendation System (Sistem Rekomendasi Pelanggan Terbaik / Siraper) is a web-based application developed to rank customers according to business-defined criteria. The system can be utilized for various purposes, including customer interest segmentation and the identification of customers who qualify for specific bonuses, incentives, or other benefits.

    This application is a Decision Support System (DSS) that implements the Simple Multi-Attribute Rating Technique Exploiting Ranks (SMARTER) method using TypeScript. Its key features include a dashboard for monitoring current customer rankings, ranking customers according to specified criteria within a defined period, managing and configuring the criteria used in the evaluation process, and maintaining customer data.
