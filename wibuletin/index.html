    Description

    Wibuletin is a web-based application designed to provide a user-friendly interface for exploring anime information through the Jikan public API. It features various search and filtering options, including tags, genres, release year, and score-based sorting.

    This project was inspired by the popular anime database website MyAnimeList and was developed as a personal learning project to gain hands-on experience working with public APIs.
