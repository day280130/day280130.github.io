    Description

    Tax Calculation Information System (Sistem Informasi Perhitungan Pajak / Siperak) is a desktop application developed to calculate Value Added Tax (Pajak Pertambahan Nilai / PPN) on products sold by a business. The system maintains transaction records and automatically calculates tax liabilities based on product sales and the applicable PPN rate, which was 11% at the time of development.

    Key features of the application include a dashboard for monitoring gross income and outstanding tax liabilities, transaction record management, product management, and role-based user account management. These features help reduce manual bookkeeping efforts while improving the accuracy and accessibility of tax-related information.
